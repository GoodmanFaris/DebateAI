# 🧠 DebateAI

AI-powered mobile application for communication, negotiation, and critical thinking training through interactive debate scenarios.

---

## 🚀 Overview

DebateAI is a full-stack mobile application designed to help users improve real-world communication skills in a safe, controlled environment.  

The app simulates realistic scenarios (e.g., negotiating salary, persuading others, handling conflict) and uses AI to evaluate user responses, provide structured feedback, and track progress over time.

---

## ✨ Features

- 🎯 AI-driven debate scenarios
- 💬 Real-time conversational interaction
- 📊 Performance scoring and feedback
- 🧠 AI coaching (premium feature)
- 🏆 Leaderboards and progress tracking (XP, streaks, levels)
- 📜 Session history and public replays
- 🔐 Secure authentication (JWT + Google OAuth)
- 🖼 Avatar upload (Cloudinary)
- 💰 Monthly premium subscription model
- 🤖 Daily scenario generation via automation (n8n)

---

## 🏗 Tech Stack

### 📱 Frontend
- React Native (Expo)
- TypeScript
- Zustand (state management)
- TanStack Query (data fetching & caching)
- Expo Secure Store (secure persistence)

### ⚙️ Backend
- FastAPI
- PostgreSQL (Neon)
- SQLModel ORM
- Alembic (database migrations)

### ☁️ Infrastructure
- Backend deployed on **Render**
- Frontend built with **EAS (Expo Application Services)**
- Media storage via **Cloudinary**

### 🤖 AI & Automation
- LLM-based evaluation and coaching
- Prompt-engineered scoring system
- Automated scenario generation using **n8n workflows** (daily content)

By combining AI simulation with structured feedback, the app enables users to:
- practice persuasion
- improve communication clarity
- build confidence in real-world situations
